# Snakemake Fast-GBS

## About

This project created a snakemake workflow for the https://academic.oup.com/gigascience/article/7/6/giy057/5001424#supplementary-data

## Requirements

- Bash 5.0.17(1)-release
- snakemake 7.25.2
- Python 3.8.8
